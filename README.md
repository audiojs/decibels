decibels
===

Convert values to and from decibels.

## Install [via npm](https://www.npmjs.com/package/decibels)

```js
$ npm install decibels
```

## API

```js
var gainToDecibels = require('decibels/from-gain')
var decibelsToGain = require('decibels/to-gain')
```

### `gainToDecibels(gain)`

### `decibelsToGain(db)`
